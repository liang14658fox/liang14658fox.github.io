## 常用API
1. Vue渲染后刷新
```js
this.$nextTick(()=>{
// 渲染后的操作
})
```
2. element刷新表单校验和值
```js
this.$refs["formName"].resetFields();
```
3. Vue强制刷新页面内容
```js
this.$forceUpdate()
```
