## 根据条件渲染class
```html
<template>
<div v-bind:class="{ active: isActive }"></div>
</template>

<script>
export default {
    data() {
        isActive: true,
    }
}
</script>
```
结果渲染为：
```html
<div class="active"></div>
```

## 可以将其定义到data()中
```html
<template>
<div :class="classObject"></div>
</template>

<script>
export default {
    data() {
        classObject:{
            active: true,
        }
    }
}
</script>
```
结果渲染为：
```html
<div class="active"></div>
```

## 使用三元表达式
```html
<template>
<div :class="isActive ? 'div_1' : 'div_2'"></div>
<div :style="{ 'height': ( isActive ? '100px' : '' ) }"></div>
</template>

<script>
	export default {
		data() {
			return {
                isActive:true,
            }
        }
    }
</script>

<style>
.div_1 {}

.div_2 {}
</style>
```

## 多个对象时可使用数组和对象数组
```html
<div :class="[class1,isActive? 'class2':'class3']"></div>
<div :class="[class1,{active2:isActive}]"></div>
<div :style="[
    { 'height': ( isActive ? '100px' : '' ) },
    { 'width': ( isActive ? '' : '100px' ) } 
]"></div>

```


## vue 循环遍历背景图
循环背景图获取index
```html
<template>
  <div id="app">
      <div 
        class='pic'  
        v-for="(item,index) in list"  
        :key=index 
        :style="{backgroundImage:'url(' + item.img+ ')', 
        backgroundRepeat:'no-repeat',backgroundSize: 'cover'}" 
        @click="handleCilck(index)">
      </div>
  </div>
</template>
```
```js
export default {
  name: 'demo',
   data(){
        return {
          list:[
            {img:require('./logo.png')},
            {img:require('./miao.jpg')},
            ]
        }
   },
   methods:{
     handleCilck(index){
        console.log(index)
     }
   }
}
```

```css
.pic{
  width: 300px;
  height: 300px;
}
```
