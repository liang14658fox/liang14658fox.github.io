  1. 下载并安装Node.js：在https://nodejs.org/en/上下载最新的Node.js安装程序，双击运行，按照提示一步步安装即可。或者通过HomeBrew安装node。

  2. 安装VitePress。

sh
yarn global add @vitepress/cli
1
  1. 创建一个名为my-project的VitePress项目。
sh
vitepress new my-project
1
  1. 进入刚才创建的项目目录。
sh
cd my-project
1
  1. 安装项目所需要的依赖包。
sh
yarn install
1
  1. 启动项目。
sh
vitepress dev
1
